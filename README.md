# Lm18-Enclosure-V8_Absolute

📌 Visão Geral
Este repositório contém o design proprietário do gabinete (case) para o sistema LMKAR (Geo-temporal Attendance System). O projeto foi desenvolvido para resolver falhas comuns em prototipagem IoT, como estresse mecânico de cabeamento e falta de ergonomia para o usuário final.

O modelo V9 abandona o padrão honeycomb em favor de paredes sólidas de alta resistência (3mm), garantindo maior durabilidade em ambientes escolares e acadêmicos.

🛠 Especificações Técnicas
Chassi: Estrutura otimizada de 120 x 100 x 60 mm.

Vão de Serviço: Torres de sustentação de 15mm para evitar fadiga em jumpers Dupont e permitir fluxo de ar interno.

Profundidade Total: Consolidada em 42mm, garantindo um perfil low-profile na parede.

IHC (Interface Homem-Computador): Inclinação ergonômica da face frontal para facilitar a leitura do LCD e o escaneamento RFID.

Assinatura Física: Gravuras técnicas em baixo-relevo (1mm) integradas ao molde (Marcação de autenticidade GLm18).

🧩 Hardware Compatível
MCU: ESP32 DevKit V1 (30/38 pins).

Display: LCD 16x2 ou 20x4 com adaptador I2C.

Sensor: Módulo RFID RC522.

🖨️ Recomendações de Impressão 3D
Material: PLA+ ou PETG (resistência superior a impactos).

Preenchimento (Infill): 20% (Padrão Gyroid recomendado para rigidez estrutural).

Paredes: Mínimo de 3 perímetros.

Suportes: Apenas nas torres internas de 15mm e no bezel do LCD.
